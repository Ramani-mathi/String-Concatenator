String Concatenator

A simple web application built using HTML, CSS, and JavaScript.

# Features
- Accepts two user inputs
- Concatenates strings using JavaScript
- Displays the output instantly
- Simple and responsive UI

# Technologies Used
- HTML5
- CSS3
- JavaScript

# How to Use
1. Enter text in the first input field.
2. Enter text in the second input field.
3. Click the Submit button.
4. View the concatenated result in the output area.

# Learning Outcomes
- DOM Manipulation
- JavaScript Functions
- Event Handling
- User Input Processing
- Basic UI Design
